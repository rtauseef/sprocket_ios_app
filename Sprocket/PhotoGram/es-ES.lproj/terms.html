CONDICIONES DE SERVICIO DE sprocket

Al usar esta aplicación, podrá conectarse a Internet y utilizar su cuenta de Instagram, Facebook o Flickr para crear e imprimir el contenido deseado.

Para usar esta aplicación, puede ser necesario conectarse a Internet. El uso inalámbrico de la aplicación depende del entorno físico y de la distancia a la que se encuentre del punto de acceso. Se pueden aplicar planes de datos comprados por separado o tarifas de uso. Los tiempos de impresión y las velocidades de conexión pueden variar. Para imprimir con dispositivos móviles iOS, necesita una impresora Airprint™. Para Android, necesita una impresora compatible con estos dispositivos. La impresora conectada de forma inalámbrica y el dispositivo móvil deben encontrarse en la misma red para que se conecten entre sí y realizar la impresión.

Si usa una red de datos para conectarse a Internet, es posible que se apliquen cargos adicionales por el uso de datos en función de su plan de red.

Para mejorar las próximas versiones de esta aplicación y de las impresoras HP, recopilaremos cada cierto tiempo los datos de uso de esta aplicación de forma anónima.
